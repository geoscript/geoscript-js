The :mod:`geom` module
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


The :mod:`geom` module provides a provides constructors for point, line,
polygon and multi-part geometries.

.. code-block:: javascript

    js> var GEOM = require("geoscript/geom");
