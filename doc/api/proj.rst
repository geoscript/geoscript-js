The :mod:`proj` module
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

The :mod:`proj` module exports a Projection constructor and methods for
transforming geometries between coordinate reference systems.

.. code-block:: javascript

    js> var PROJ = require("geoscript/proj");
